# UI Constants

Breakpoints, limits, and dimension values used across the frontend.

## Breakpoints

| Name | Value | Usage |
|------|-------|-------|
| Mobile/Desktop | 768px | DataTable card/table view switch |
| Sidebar | 1024px | Desktop sidebar vs mobile navigation |

## Pagination

| Setting | Value |
|---------|-------|
| Page size options | 10, 20, 30, 50, 100 |
| Default page size | 100 |
| Custom tables page size | 50 |

## Row Limits

| Context | Limit |
|---------|-------|
| DataTable default | 10,000 |
| Dashboard components | 1,000 |
| Query preview | 1,000 |

## Layout Dimensions

| Element | Value |

| Sidebar expanded | 256px |
| Sidebar collapsed | 56px |
| Container max-width | 1600px |
| Padding mobile | 16px |
| Padding desktop | 24px |
